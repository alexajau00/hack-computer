 ## Hack-Computer 

In this project, I implemented the key components of a Hack computer system. This project includes hardware modules written in Hardware Description Language (HDL) and software components written in C to parse and process Hack assembly instructions. The Hack computer architecture is used to demonstrate how hardware and software interact. The assembler processes assembly instructions, manages symbolic references through a symbol table, and converts the instructions into structured representations that correspond to machine level operations. 

Through this project, I gained hands-on experience with computer architecture, digital design logic, low-level programming, and parsing techniques, while also gaining a deeper understanding of how software tools interact with hardware systems.
